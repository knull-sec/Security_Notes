# Authenticaion
### **Authentication**

- **Definition:** The means of proving *who you are*.
- **Question:** "Who are you?"
- **Goal:** Verifying identity (e.g., logging in).

### **Authorization**

- **Definition:** Determining the permissions valid for a specific identity.
- **Question:** "What can you access?"
- **Goal:** Verifying access rights (e.g., admin vs. standard user privileges).
